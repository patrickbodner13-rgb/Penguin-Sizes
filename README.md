# Penguin-Sizes
This is an analysis of three species of penguins living on three different islands: Biscoe, Dream and Torgensen. The analysis compares culmen and body mass of the species.

🐧 Penguin Data Analysis Dashboard
📌 Overview

This project explores physical characteristics of penguins across different islands and species using the Palmer Penguins dataset.

The goal was not just to visualize the data, but to identify what actually drives observed differences in size and morphology.

The dashboard focuses on answering the question:

Where do penguins with the largest body mass and longest culmen and flipper measurements live—and why?

🎯 Key Objectives
Analyze differences in penguin size across islands
Identify whether geography or species drives these differences
Build a dynamic dashboard that allows users to explore underlying drivers
Present insights clearly and with analytical rigor

🔍 Key Insights
1. Island-Level Observation
Penguins on Biscoe Island show the highest:
Body mass (~4.72 kg)
Flipper length (~209.56 mm)
Culmen length (~45.25 mm)
2. Species as the Primary Driver
These differences are not driven by geography alone
Gentoo penguins, which are significantly larger, are primarily located on Biscoe

👉 This explains the higher averages observed on that island

3. Morphological Differences by Island
Penguins on Torgersen Island show the highest culmen depth (~18.45 mm)
This is driven by the presence of Adelie penguins, indicating different physical adaptations
4. Gender Differences
Across all species, male penguins are consistently heavier than females
The magnitude of this difference varies by species
5. Correlation Between Size Metrics
A strong positive relationship exists between:
Flipper length
Body mass

The scatter plot clearly shows distinct clustering by species, reinforcing that:

Species composition is the dominant factor behind size differences

🧠 Key Takeaway

Apparent differences in penguin size across islands are primarily driven by species composition rather than geography alone.

This highlights the importance of:

Controlling for confounding variables
Avoiding conclusions based solely on aggregated data
